###  [1일차 - git init](https://www.youtube.com/watch?v=fCY1t3QSEhw&list=PLuHgQVnccGMA8iwZwrGyNXCGy2LAAsTXk&index=6)

***

- `git init`  : 현재 디렉토리를 버전관리 하겠다는 사실을 git에게 알려주는 명령어. 
  - `git init` 명령어를 실행하면, 해당 디렉토리 내부에 `.git ` 디렉토리가 생성된다.
